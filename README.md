# Cloud-Based LinkedIn Reminder Automation

This project implements a cloud-based automation workflow using GitHub Actions
to send scheduled email reminders for LinkedIn content posting.

## How it works
- Runs automatically at 9:00 AM and 7:00 PM IST
- Selects content folders based on date and time
- Sends reminder emails with captions and image references
- Requires no local system to be online

## Tech Stack
- Node.js
- GitHub Actions
- Gmail SMTP (App Passwords)

## Status
Used for managing a 25-post LinkedIn content series.
